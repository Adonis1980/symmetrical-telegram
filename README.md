## ğŸ§± Architecture Overview
The system is designed around 4 components:

1. **Agent Layer**  
   Logical units that perform tasks such as outreach, scraping, analysis, and sequencing.

2. **Action Layer**  
   Sends emails, texts, runs API calls, creates CRM records.

3. **Memory Layer**  
   Logs history, updates lead status, stores conversation states.

4. **Interface Layer**  
   Connects UI/command tools like Telegram, webhooks, or dashboards.
